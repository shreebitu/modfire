## 🌐 Website Overview

ShreeBitu.in is a cloud-based file library and download platform where users can explore, download, and manage software files including APKs, Linux ISOs, Windows tools, open-source projects, presentations, PDFs, and ZIP archives.

## 🗂️ Data Files (JSON)

All content is loaded dynamically via `assets/js/dynamicLoader.js` from:

```
data/
├── apk.json        → APK downloads
├── linux.json      → Linux ISOs
├── windows.json    → Windows software
├── opensource.json → Open source links
├── ppt.json        → Presentations
├── pdf.json        → PDF documents
└── zip.json        → ZIP archives
```

Each JSON entry format:
```json
{
  "title": "App Name",
  "image": "path/to/image",
  "description": "Short description",
  "link": "page/file.html",
  "date": "YYYY-MM-DD",
  "size": "XX MB"
}
```
