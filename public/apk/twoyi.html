What is Twoyi?

Twoyi is a lightweight Android container that runs a nearly complete Android system as a normal app — no root, no bootloader unlock needed. Made with ❤️ by weishu.

It allows you to use Taichi·Yang, Xposed, EdXposed & LSPosed without unlocking the bootloader. Boots in under 3 seconds!

Key Features

  • Ultra-Fast Boot: Twoyi boots in under 3 seconds (after initial setup).
  • No Bootloader Unlock: Use Xposed frameworks without unlocking.
  • Root Features: Use root access on non-rooted devices safely.
  • Virtual HAL: Implement virtual camera and virtual sensors easily.

Note: Twoyi is open source. Make sure you enable virtualization for the best performance.
