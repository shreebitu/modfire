What is HAPI?

HAPI is like a smart remote control for your AI. It lets you run powerful tools like Claude Code, Gemini, or OpenCode on your computer, but gives you the power to control them from your phone or any browser. If you're a developer who wants to stay connected to your work without being glued to your desk, HAPI is your new best friend.

Frequently Asked Questions

The Basics

What exactly is HAPI?

Think of it as a remote control for your AI agents. You run the heavy work on your computer, but HAPI lets you talk to it, see what it's doing, and approve actions from your phone—anywhere you go.

Is it really free?

Yep! HAPI is 100% open source and free to use forever.

Why the name "HAPI"?

It's a play on the word "Happy". We believe AI coding should be a happy, flexible experience that doesn't keep you glued to your desk.

Using HAPI

How do I approve permissions remotely?

When your agent wants to edit a file, you'll get a notification on your phone. Just open the HAPI app, see the request, and tap "Approve". It's that simple.

Can I use voice commands?

Absolutely. If you have an ElevenLabs key, you can hit the microphone button and literally talk to your code. "Hey Claude, refactor this function" works like a charm.

Does it support Windows terminal?

Remote terminal access works on macOS and Linux. Windows support is coming soon as the underlying tech matures!

Safety & Privacy

Is my code private?

100%. HAPI is local-first, meaning your data stays on your machine. We don't upload your code to any servers.

How secure is the connection?

The relay uses WireGuard + TLS for end-to-end encryption. Your data is encrypted directly from your phone to your computer.

🔄 Permission Flow

  1. AI agent requests tool permission
  2. CLI sends request to Hub
  3. Hub notifies you via Phone/Telegram
  4. You approve on your phone
  5. Hub relays decision to CLI

🚀 Message Flow

  1. You send message from phone
  2. Hub receives and emits to CLI
  3. CLI passes to AI Agent
  4. Agent processes and streams back
  5. Hub pushes live updates to you

Local vs Remote Modes

HAPI's defining feature is the ability to seamlessly hand off control. When you're at your desk, use the Local Mode (Terminal) for native speed. When you step away, Remote Mode (Phone) takes over automatically without losing session state.

Cursor Agent Setup

HAPI supports Cursor Agent CLI for running Cursor's AI coding agent with full remote control via web and phone.

🛠️ Prerequisites

Install the Cursor Agent CLI first:

# macOS / Linux curl https://cursor.com/install -fsS | bash
# Windows (PowerShell) irm 'https://cursor.com/install?win32=true' | iex

Verify it's working: agent --version

🚀 Usage Commands

hapi cursor Start new agent session
hapi cursor --continue Resume most recent chat
hapi cursor --mode plan Start in Plan mode
hapi cursor --yolo Bypass approval prompts

⚖️ Permission Modes

Mode Description
default Standard agent behavior
plan Plan mode - design before coding
ask Explore code without edits
yolo Bypass all approval prompts

Tip: You can change the mode from the web UI during a live session.

🏠 Local Mode

Run from your terminal. Full interactive experience with instant keyboard feedback.

🌍 Remote Mode

Spawn from web/phone. Uses --trust to run tools via secure background processes.

⚠️ Limitations
  • Remote mode uses --trust (no per-request approval)
  • Use agent ls to find previous Chat IDs for manual resume
🔗 Integration
  • Monitor live activity in Web & Telegram
  • Messages queue automatically when switching modes

Setup Instructions

  1. Get an API Key from elevenlabs.io
  2. Set the environment variable:
    export ELEVENLABS_API_KEY="your-api-key"
  3. Start the hub with hapi hub --relay
  4. In the web app, click the microphone button to start talking!

Voice Commands Example

  • "Ask Claude to refactor the auth module"
  • "Go ahead and run the tests" (Approve)
  • "No, don't delete that file" (Deny)

CLI (Required) Wraps AI agents (Claude/Gemini/etc) and runs sessions.
Hub (Required) Central coordinator for persistence and remote access.
Runner (Optional) Background service for remote session spawning.

1. Prerequisites

Ensure you have one of these AI agents installed on your machine:

# For Claude Code: claude --version
# For Gemini CLI: gemini --version
# For OpenCode CLI: opencode --version

2. Install the CLI

npm install -g @twsxtd/hapi --registry=https://registry.npmjs.org

Note: Use the official npm registry for global installs to ensure platform packages sync correctly.

3. Hub Setup (Choose your way)

Public Relay (Recommended) hapi hub --relay

Displays a QR code for instant phone access. Fully E2E encrypted via WireGuard + TLS.

Local Only hapi hub

Listens on http://localhost:3006. Perfect for desktop use without remote access.

4. Background Runner (Optional)

Run as a service to spawn sessions from your phone without keeping a terminal open:

hapi runner start hapi runner status
📱 Telegram Mini App

Set your TELEGRAM_BOT_TOKEN and HAPI_PUBLIC_URL, then message your bot with /start.

🎙️ Voice Assistant

Enable hands-free control by setting your ELEVENLABS_API_KEY before starting the hub.
